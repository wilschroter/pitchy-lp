# Pitchy LP

Mobile landing page for [Pitchy](https://github.com/wilschroter/pitchy). Static HTML, deployed on Render.

## Deploy

Auto-deploys on every push to `main` via `render.yaml`. Render service: `pitchy-lp` (free tier, static site, Oregon region).

## Local preview

Open `index.html` in a browser. No build step.

## Structure

- `index.html` — the artboard (3 mobile screens of the LP funnel)
- `static/` — image assets
- `render.yaml` — Render static-site config
